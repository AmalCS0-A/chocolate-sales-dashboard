# Chocolate Sales Dashboard 

An interactive Streamlit dashboard for exploring and analyzing chocolate sales transactions through a complete data analysis pipeline.

## Live Demo

https://chocolate-sales-dashboard-wcfvdetskrpzeetshtzrwq.streamlit.app/

---

## Overview

This project analyzes sales data from a chocolate distribution company between 2022 and 2024.  
The dashboard combines data preprocessing, exploratory analysis, visualization, and regression modeling inside an interactive web application built with Streamlit.

The goal of the project is to transform raw transactional data into meaningful business insights through clear visual analytics and predictive modeling.

---

## Main Features

### Data Processing
- Cleaning and formatting raw sales data
- Date parsing and feature engineering
- Missing value and duplicate detection

### Exploratory Analysis
- Revenue distribution analysis
- Country-based sales comparison
- Monthly revenue trends
- Product performance analysis
- Correlation heatmaps
- Interactive visualizations

### Machine Learning
Implementation and comparison of:
- Linear Regression
- Ridge Regression
- Lasso Regression

Evaluation metrics:
- MAE
- RMSE
- R² Score
